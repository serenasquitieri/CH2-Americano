# CH2-Americano

Lucky Locked
A secure iOS password manager built with SwiftData and Face ID.

Lucky Locked is a password management application inspired by Apple's "Passwords" app, focusing on an improved password collection system and secure initial entry using Face ID.

Features:

Face ID Entry: Secure and immediate access to your vault.

Improved Collection: A streamlined interface for adding new passwords and categories.

Persistent Storage: All data is saved securely on-device using SwiftData.

Technology Used:

SwiftUI

SwiftData

LocalAuthentication
